Dado un texto, cuenta el número de palabras que contiene.

Input Format

Un string con varios saltos de línea. El texto termina con la palabra END (que no debe procesarse)

Constraints

1 <= L <= 100

Output Format

Un entero indicando el número de palabras

Sample Input 0

hola mundo!
END

Sample Output 0

2

Sample Input 1

hola, que tal.
END

Sample Output 1

3

Sample Input 2

Lorem ipsum dolor sit amet,
consectetur adipiscing elit.
END

Sample Output 2

8

Sample Input 3

Lorem ipsum
dolor sit amet,
consectetur adipiscing
elit.
END

Sample Output 3

8
